# Swift Stay

## Overview

**Swift Stay** is a luxury vacation rental platform designed to offer premium travel experiences. Built with modern web technologies, it allows users to discover exclusive properties, manage bookings, and host unique stays.

## Features

- **Luxury Listings**: Curated collection of high-end resorts, villas, and palaces.
- **Smart Search**: Find the perfect getaway by destination, date, and guest count.
- **Secure Authentication**: Google and email-based login secure by JWT.
- **Booking Management**: Seamless booking process for users.
- **Host Dashboard**: Manage property listings and availability.
- **Responsive Design**: Optimized for all devices with a premium aesthetic.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Shadcn UI, Vite
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT, Google OAuth
- **Storage**: Cloudinary (Image management)



### Prerequisites

- Node.js (v14+)
- Yarn or npm
- MongoDB URI
- Cloudinary Account
- Google Cloud Console Project (for OAuth)
